# ext-websocket manual

Documentation site for `ext-websocket`, a native PHP extension providing RFC 6455 WebSocket protocol helpers and a callback-based server runtime.

The site is static and can be served directly by GitHub Pages.

Covered topics:

- Installation and quick-start server setup.
- `WebSocket\Server`, `WebSocket\ServerOptions`, `WebSocket\Connection`, and `WebSocket\Protocol` reference.
- Server limits, handshake and idle timeouts, UTF-8 validation, close-code validation, and random mask keys.
- Production guidance and benchmark links.
